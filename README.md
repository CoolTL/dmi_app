# DMI app

# Model
Model findes i model folderen. Model classen har 5 metoder, en der har nogle variabler, og de andre fem henter og behandler dataet. Den sidste returnerer dataen og akserne til at være plottet.

# View
View findes i view folderen. View classen har 2 metoder, en der initialiser GUIen, og en der plotter det nye data.

# Controller
Controller findes i controller folderen. Controller classen har 2 metoder, en der forbinder knappens signal til et slot, og en der køres når man trykker på knappen.

# Dependencies
* Python 3.13
* PySide6
* Matplotlib
* Pandas
* requests
* datetime
